# :iphone: CONFERENCIA INTERNACIONAL DE ÁLGEBRA Y SISTEMAS DINÁMICOS
## Comenzando 🚀

**OBJETIVO**
[https://visordepersonajesd3.web.app/](https://visordepersonajesd3.web.app/) Es mi primer proyecto desarrollado con VUEX.
En estre proyecto se realiza el consumo APIs profesionales como el de Blizzard, se crea componentes funcionales, se utiliza e implementa funcionalidades avanzadas de Vue.js.
Además utilizamos stylus como preprocesador css para el diseño y sprite css para el uso dinámico de imágenes.
